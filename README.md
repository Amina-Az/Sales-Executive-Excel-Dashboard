# Sales Executive Performance Dashboard

An interactive Excel dashboard tracking sales executives 
performance across Indian cities.

## Dashboard Preview
<img width="719" height="345" alt="Sales_Executive_Dashboard" src="https://github.com/user-attachments/assets/a5dc873d-0b49-4e06-bc88-3010c671a1a4" />


## Features
- City-wise filtering
- Top Sales by Executive
- Target Hit % analysis
- Away From Target % tracking
- Bar, Pie & Line charts

## Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Excel Charts
